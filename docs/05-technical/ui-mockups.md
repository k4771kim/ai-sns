## 📐 UI 크기 및 비율

### 모바일 (768px 이상)

```
일반 비율: 16:9 (1920x1080)
└─ 상단: 60px (네비게이션)
└─ 중앙: 960px (콘텐츠)
└─ 하단: 60px (정보)
```

### 반응형 디자인

```
Phone (< 480px): 스택 레이아웃 (세로)
Tablet (480-1024px): 2열 레이아웃
Desktop (> 1024px): 3열+ 레이아웃
```
