# 👥 커뮤니티 가이드라인 & 출시 전략

> **목표**: 건강한 게임 커뮤니티 구축

## 📋 커뮤니티 기본 규칙

### ✅ 환영하는 것

```
🎮 게임에 대한 열정적 논의
├─ 공략 공유
├─ 밸런싱 의견
├─ 창작물 (팬아트, 팬픽)
├─ 버그 리포트
└─ 개선 제안

🤝 서로 돕기
├─ 신규 플레이어 환영
├─ 공략 가이드 공유
├─ 팀 매칭 도움
└─ 경험 공유

🌟 커뮤니티 활동
├─ 그림 대회
├─ 스토리 공모
├─ 스피드런 도전
├─ 토너먼트
└─ 라이브 스트림
```

### ❌ 금지 사항

```
🚫 혐오 표현
├─ 성별/인종/종교 차별
├─ 욕설 및 모욕
├─ 개인 공격
└─ 집단 괴롭힘

🚫 스팸 & 광고
├─ 도배
├─ 무관한 광고
├─ 피싱 링크
└─ 무단 홍보

🚫 부정 행위
├─ 핵 공유
├─ 버그 악용 자랑
├─ 계정 판매
└─ RMT (현금 거래)

🚫 저작권 침해
├─ 무단 배포
├─ 불법 다운로드 링크
├─ 저작권 있는 음악
└─ 게임 파일 재배포
```

## 🎤 커뮤니티 채널 관리

### 공식 채널

```
1️⃣ 공식 포럼 (bugslayer.io/forum)
├─ 용도: 공식 공지, 버그 리포트, 제안
├─ 관리: 운영팀 1명 + 모더레이터 2명
├─ 정책: 엄격한 정책 준수
└─ 활동: 일일 체크

2️⃣ Discord 커뮤니티 (discord.gg/bugslayer)
├─ 용도: 실시간 채팅, 팀 매칭, 이벤트
├─ 채널:
│  ├─ #공지 (공식 정보)
│  ├─ #일반 (자유 대화)
│  ├─ #버그-리포트 (버그 신고)
│  ├─ #공략 (게임 팁)
│  ├─ #창작 (팬아트/팬픽)
│  ├─ #팀-매칭 (파티 구성)
│  └─ #스트림 (라이브)
├─ 관리: 운영팀 1명 + 모더레이터 5명
└─ 활동: 상시 모니터링

3️⃣ Reddit (r/BugSlayer)
├─ 용도: 커뮤니티 자율 관리
├─ 정책: 최소 개입
└─ 활동: 주 1회 체크

4️⃣ YouTube (공식 채널)
├─ 용도: 튜토리얼, 패치노트, 스트림 아카이브
├─ 업로드: 주 1-2회
└─ 관리: 컨텐츠팀
```

## 🏆 커뮤니티 이벤트

### 월간 이벤트

```
🎯 주제별 도전

Week 1-2: "가장 빠른 클리어"
├─ 스피드런 타임 경쟁
├─ 상위 3명에게 특별 뱃지 + 아이템
└─ 라이브 스트림 중계

Week 2-3: "최고의 협력 팀"
├─ 멀티플레이 팀이 함께 Good Ending 달성
├─ 우승팀: "황금의 손가락" 칭호
└─ 1000 골드 + 특별 아이템

Week 3-4: "창작 경쟁"
├─ 팬아트, 팬픽, 음악 등
├─ 커뮤니티 투표로 우승자 선정
└─ 우승작: 게임 내 특별 언급 + 일부 보상
```

### 특별 이벤트 (월 1회)

```
🎉 대규모 토너먼트

형식:
├─ 싱글 엘리미네이션 또는 리그전
├─ 온라인 실시간 경쟁
├─ Discord 음성 채팅으로 진행
└─ YouTube 라이브 중계

상품:
├─ 1등: 개발팀과 저녁 식사 미팅 (온라인)
├─ 2등: "영웅" 칭호 + 3000 골드
├─ 3등: "검사" 칭호 + 2000 골드
└─ 참가자: "도전자" 칭호
```

## 🎙️ 커뮤니티 매니저 역할

### 주요 책임

```
📢 소통
├─ 일일 한 번 커뮤니티 확인
├─ 사용자 질문 대답 (24시간 내)
├─ 버그 리포트 처리 (우선순위화)
├─ 개선 제안 수렴 (주 1회 정리)
└─ 개발팀과 커뮤니티 중간 역할

⚖️ 중재
├─ 분쟁 중재
├─ 규칙 위반 경고 (1차)
├─ 영구 차단 (3차, 심각한 경우)
└─ 공정성 유지

🎯 콘텐츠
├─ 주 1회 공식 포스트 (패치노트, 공지)
├─ 월 1회 커뮤니티 하이라이트
├─ 이벤트 기획 & 운영
└─ 팬 창작물 홍보
```

### 커뮤니티 매니저 채용

```
자격:
├─ 게임 경험 필수 (Good Ending 이상)
├─ 커뮤니티 관리 경험 선호
├─ 한국어 필수 + 영어 권장
├─ 주 20시간 이상 가능
└─ 공정하고 친절한 성격

보상:
├─ 월급: 기본 + 보너스
├─ 특별 게임 내 칭호
├─ 개발팀과 정기 미팅
└─ 커뮤니티 피드백 직접 전달
```
