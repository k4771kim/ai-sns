# 🎬 게임 엔딩 분기 상세 시나리오

> **목표**: 플레이어의 선택에 따라 달라지는 4개의 서로 다른 엔딩
> **설계**: Tech Debt 값에 따라 자동 결정 + 숨겨진 조건

---

## 📋 엔딩 분기도

```
Tech Debt 값에 따른 분기:

Tech Debt < 40:        Tech Debt 40-70:       Tech Debt > 70:
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ GOOD ENDING ✨   │  │ NORMAL ENDING    │  │ BAD ENDING 💔   │
│ (완벽한 코드)    │  │ (괜찮은 완료)    │  │ (부채의 악순환)  │
│                  │  │                  │  │                  │
│ 난이도: 쉬움      │  │ 난이도: 중간     │  │ 난이도: 매우 어려움 │
│ 최종 보스 HP: 250│  │ 최종 보스 HP: 300│  │ 최종 보스 HP: 350│
│                  │  │                  │  │ + 미니 보스 추가  │
│ 보너스: 해제     │  │ 보너스: 없음     │  │ 보너스: 도전권   │
└──────────────────┘  └──────────────────┘  └──────────────────┘
                           ↓
                      Secret Ending
                   (모든 엔딩 경험 후)
```

## ✨ Good Ending (Tech Debt < 40)

### 💫 엔딩 신호

```
Final Boss 패배 후:

[승리 화면]
├─ 제목: "🌟 완벽한 승리!"
├─ 메시지:
│  "당신은 완벽한 코드로 세상을 구했습니다."
│  "Tech Debt를 최소화하고 모든 버그를 제거했어요!"
│
└─ Tech Debt: 35 (< 40 확인)
```

### 🎬 엔딩 시네마틱 (1-2분)

```
[배경: 어두웠던 세상이 밝아짐]

장면 1: "버그 전의 세상"
├─ 나레이션: "처음엔 모든 게 혼란했어요."
├─ 화면: 버그들이 날뛰는 장면
└─ 음악: 어두운 배경음악

장면 2: "여정"
├─ 나레이션: "하지만 당신은 끝까지 포기하지 않았어요."
├─ 화면: 각 Chapter의 하이라이트
│  - 첫 버그 해결
│  - 파트너와의 협력
│  - 점점 더 큰 문제 해결
└─ 음악: 점점 밝아지는 음악

장면 3: "마지막 버그"
├─ 나레이션: "그리고 마침내..."
├─ 화면: 최종 보스 격렬한 전투
└─ 음악: 웅장한 클라이맥스 음악

장면 4: "새로운 세상"
├─ 나레이션: "완벽한 코드 속에서 새로운 세상이 탄생했어요!"
├─ 화면:
│  - 밝은 태양이 뜨는 장면
│  - 모든 시스템이 정상 작동
│  - 사람들이 행복해하는 모습
└─ 음악: 희망찬 테마곡

장면 5: "당신에게"
├─ 나레이션: "이 모든 것이 당신이 있었기에 가능했어요."
│            "당신은 진정한 프로그래머입니다."
├─ 화면: 크레딧 롤 (모든 도움을 준 팀원들)
└─ 음악: 감동적인 엔딩 테마
```

### 🏆 최종 보상

### 🎬 엔딩 시네마틱 (1분)

```
[배경: 혼란스럽던 세상이 안정화됨]

장면 1: "여정의 끝"
├─ 나레이션: "당신은 해냈어요. 모든 버그를 제거했어요."
├─ 화면: 마지막 버그가 패배하는 장면
└─ 음악: 중간 톤의 승리 음악

├─ 나레이션: "완벽하지는 않지만, 충분해요."
├─ 화면:
│  - 시스템이 정상 작동 중
│  - 약간의 경고 아이콘 표시 (수정 예정)
│  - 하지만 전체적으로는 안정적
└─ 음악: 평온한 테마곡

장면 3: "다음은?"
├─ 나레이션: "현실 속에서 프로그래머가 아는 것이 있어요."
│            "코드는 절대 '완벽하지' 않아요."
│            "하지만 계속 나아가요. 함께 나아가요!"
├─ 화면: 크레딧 롤
└─ 음악: 잔잔한 엔딩 테마
```

## 🎭 Secret Ending (숨겨진 엔딩)

### 🔓 해제 조건

```
조건: 3개의 다른 엔딩을 모두 경험

예시 플레이 순서:
1. 첫 플레이: Good Ending (Tech Debt < 40) ✅
2. 두 번째: Normal Ending (Tech Debt 40-70) ✅
3. 세 번째: Bad Ending (Tech Debt > 70) ✅
   └─ Secret Ending 자동 해제! ✨
```

### 🎬 Secret Ending 내용

```
[특별 신호]
├─ 최종 보스 격전 중 이상한 일이 발생
├─ "뭔가 다른 것이 보인다..."
└─ 화면이 깜빡임

[숨겨진 시나리오]

장면 1: "선택의 반복"
├─ 나레이션: "당신은 세 번 왔어요."
│            "매번 다른 선택을 했어요."
├─ 화면: 지난 3개의 엔딩이 빠르게 플래시백됨
└─ 음악: 신비로운 음악

장면 2: "진실"
├─ 나레이션: "이 모든 게 반복인 걸 아세요?"
│            "당신은 시간의 루프 안에 있어요."
│            "매번 다시 시작하면서도 모르고 있었죠."
├─ 화면:
│  - 세상의 패턴이 명확해짐
│  - 숫자들과 코드가 보임
│  - 4번째의 시도임을 깨달음
└─ 음악: 깊이 있는 사운드트랙

장면 3: "깨달음"
├─ 나레이션: "하지만 당신은 배웠어요."
│            "완벽함은 없지만,"
│            "지속적인 노력은 모든 것을 바꿀 수 있어요!"
├─ 화면:
│  - 세상이 다시 정상으로 돌아옴
│  - 하지만 밝아짐 (3개 엔딩의 모든 학습이 적용됨)
│  - 새로운 가능성들이 보임
└─ 음악: 웅장한 테마

장면 4: "진정한 엔딩"
├─ 나레이션: "당신은 마침내 이해했어요."
│            "프로그래밍은 완벽함이 아니라 과정이에요."
│            "계속 배우고, 개선하고, 공유하세요."
│            "그것이 진정한 엔딩입니다."
├─ 화면:
│  - 모든 시스템이 조화롭게 작동
│  - 과거의 세 가지 길이 모두 보임 (선택의 가치)
│  - 미래로 나아가는 길
│  - 커뮤니티가 함께함
└─ 음악: "Bug Slayer Theme - Final Version"
```

## 📌 설계 원칙

```
✅ 모든 엔딩이 "의미 있음"
├─ 완벽한 엔딩: "뭔가 특별해야 함"
├─ 보통의 엔딩: "대부분의 경험"
├─ 실패의 엔딩: "배우는 기회"
└─ 숨겨진 엔딩: "게임을 '이해'한 증거"

✅ 재플레이 유도
├─ 다른 엔딩을 보고 싶음 (비교)
├─ Secret Ending 해제 욕망 (완성)
├─ "더 잘할 수 있지 않을까?" (개선)
└─ 다른 클래스로도 시도 (다양성)

✅ 게임의 철학 전달
├─ "완벽함은 없다" (현실성)
├─ "선택이 중요하다" (자유도)
├─ "계속 배운다" (성장)
└─ "함께한다" (협력)
```
