## 📊 스킬 특징 분류

### 🔥 공격 스킬 (ATK 중심)
```
Frontend Mage:
├─ console.log (약점)
├─ CSS 리셋 (강화)
├─ Webpack 최적화 (속도)
└─ Promise 연쇄 (지속)

Backend Knight:
├─ Database 쿼리 (회복)
├─ API 요청 (저하)
└─ 로드 밸런싱 (다중)
```

### 🛡️ 방어 스킬 (DEF 중심)
```
Backend Knight:
├─ 트랜잭션 처리 (피해방어)
├─ 캐싱 (방어강화)
└─ 마이그레이션 (버프유지)

DBA Healer:
├─ 정규화 (방어강화)
└─ 파티셔닝 (독립)
```

### ⚡ 속도 스킬 (SPD 중심)
```
Frontend Mage:
├─ Webpack 최적화 (2배 행동)
└─ Shadow DOM (회피)

DevOps Ranger:
├─ 배포 자동화 (강화)
├─ 모니터링 (예측)
└─ CI/CD (자동)
```

### 💚 지원 스킬 (SPT 중심)
```
DBA Healer:
├─ SQL 수복 (전체회복)
├─ 락 해제 (상태해제)
└─ 백업 복구 (부활)

DevOps Ranger:
├─ 배포 자동화 (강화)
└─ 헬스 체크 (진단)
```
