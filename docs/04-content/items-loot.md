## 📋 아이템 종류

### 1. 장비 아이템 (Equip)

```
희귀도: Common → Rare → Epic
효과: 스탯 증가
교체: 더 좋은 아이템 발견 시 자동 교체
```

### 2. 소비 아이템 (Consumable)

```
타입: 물약, 스크롤, 음식
일회용: 사용 후 사라짐
전투 중 사용 가능
```

### 3. 특수 아이템 (Special)

```
수집용 또는 퀘스트용
판매 불가
히든 엔딩 관련
```

## 🎲 드롭 시스템

### 드롭 확률 계산

```
기본 공식:
드롭 확률 = (기본율) × (클래스 보너스) × (Tech Debt 보정)

클래스 보너스:
├─ Backend Knight (DEF 높음): 방어 아이템 +50%
├─ DBA Healer (SPT 높음): 회복 아이템 +50%
├─ Frontend Mage (ATK 높음): 공격 아이템 +50%
└─ DevOps Ranger (SPD 높음): 속도 아이템 +50%

Tech Debt 보정:
├─ Tech Debt < 30: 희귀 드롭 +25%
├─ Tech Debt 30-70: 정상
└─ Tech Debt > 70: 일반 드롭 +50% (좋은 아이템 드물게)
```

### 보증 시스템

```
연속 드롭 없는 경우:
├─ 10회 연속 일반 드롭: 다음 희귀 드롭 보증
├─ 30회 연속 희귀 이하: 다음 에픽 드롭 보증
└─ 보증 적용 시 보상 2배
```

## 🎯 인벤토리 관리

### 제한사항

```
장비 슬롯: 4개 (머리, 몸, 손, 발)
소비 슬롯: 20개
총 용량: 24개

오버플로우 시:
├─ 자동으로 가장 약한 Common 팔기
├─ 또는 플레이어 선택
└─ 특수 아이템은 자동 팔지 않음
```

### 정렬 옵션

```
- 희귀도 순
- 효과 크기 순
- 획득 순서
- 판매가 순
```

