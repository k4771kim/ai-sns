## 1. 밸런싱 원칙

### ⚖️ 포인트 시스템

```
모든 클래스: 총 28 포인트 (균등 배분)
├─ ATK (공격력): 4-10
├─ DEF (방어력): 4-10
├─ SPD (속도): 4-10
└─ SPT (지원력): 4-10

포인트 합계: 28점 (모두 동일)
```

### 🎯 역할별 특징

```
공격 (ATK) 중심: ATK 9-10, 다른 스탯 낮음
탱커 (DEF) 중심: DEF 9-10, 속도 낮음
서포터 (SPT) 중심: SPT 9-10, 공격력 낮음
스피드 (SPD) 중심: SPD 9-10, 방어력 낮음
```

### 💡 밸런싱 철학

```
"모든 클래스는 상황에 따라 최고가 될 수 있다"

예시:
├─ 1v1 대결: Frontend Mage 최고
├─ 난관 대비: Backend Knight 최고
├─ 장시간 전투: DevOps Ranger 최고
└─ 긴급 상황: DBA Healer 최고
```

## 2. 4개 기본 클래스 재정의

### 🔧 스탯 재계산

#### 1️⃣ Frontend Mage (프론트엔드 마법사)
**역할**: 공격 스페셜리스트 | **난이도**: ⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 10 (최고 공격력)
├─ DEF: 3  (낮은 방어)
├─ SPD: 10 (최고 속도)
└─ SPT: 5  (약한 지원)

합계: 28점 ✅

특징:
├─ 장점: 빠르고 강함 (번개같은 공격)
├─ 단점: 매우 약함 (한 대에 죽을 수 있음)
├─ 플레이 스타일: "먼저 때리는 자가 우승" (first strike wins)
└─ 난이도: 초급 (직관적이지만 조심해야 함)
```

---

#### 2️⃣ Backend Knight (백엔드 나이트)
**역할**: 탱커 | **난이도**: ⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 5  (중간 공격)
├─ DEF: 10 (최고 방어)
├─ SPD: 3  (느림)
└─ SPT: 10 (최고 지원)

합계: 28점 ✅

특징:
├─ 장점: 강하고 파티를 지킴
├─ 단점: 느려서 선제공격 못 함
├─ 플레이 스타일: "방어와 회복의 벽"
└─ 난이도: 초급 (안정적)
```

---

#### 3️⃣ DevOps Ranger (DevOps 레인저)
**역할**: 속도 서포터 | **난이도**: ⭐⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 6  (중간 공격)
├─ DEF: 5  (중간 방어)
├─ SPD: 10 (최고 속도)
└─ SPT: 7  (좋은 지원)

합계: 28점 ✅

특징:
├─ 장점: 균형잡힌 다재다능함
├─ 단점: 특출난 게 없음
├─ 플레이 스타일: "유연한 전술가"
├─ 난이도: 중급 (여러 역할을 해야 함)
└─ 멀티플레이 추천: ⭐⭐⭐
   (적응력으로 아무 조합과도 잘 맞음)
```

---

#### 4️⃣ DBA Healer (DBA 힐러)
**역할**: 지원 스페셜리스트 | **난이도**: ⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 3  (최저 공격)
├─ DEF: 4  (낮은 방어)
├─ SPD: 6  (중간 속도)
└─ SPT: 15 (최고 지원)

합계: 28점 ✅

특징:
├─ 장점: 파티 전체 살림
├─ 단점: 혼자는 매우 약함
├─ 플레이 스타일: "생존과 회복의 여신"
├─ 난이도: 초급 (직관적)
└─ 멀티플레이 필수: ⭐⭐⭐
   (파티가 없으면 불가능)
```

## 3. 4개 숨겨진 클래스 미리보기

### 🎁 숨겨진 클래스 전략

```
발견 방법:
├─ GPU Warlock: "병렬 처리 챌린지" (3턴 내 2배 데미지)
├─ Cat Summoner: "귀여움 모드" (1시간 연속 플레이)
├─ Code Ninja: "스피드런" (5분 내 Chapter 1 클리어)
└─ Light Mage: "완벽 클리어" (Tech Debt 0으로 전체 완주)
```

### 🚀 GPU Warlock (GPU 워록) - 병렬 처리 마법사

**역할**: 공격 + 다중 타겟 | **난이도**: ⭐⭐⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 8  (높은 공격)
├─ DEF: 2  (매우 약함)
├─ SPD: 11 (초고속)
└─ SPT: 7  (좋은 지원)

합계: 28점 ✅

핵심 스킬:
├─ "병렬 처리": 모든 적에게 동시 공격
├─ "GPU 가속": 다음 턴 행동 2배 빠름
├─ "쿨링 부족": 매 3턴마다 1턴 스턴
└─ 특징: 빠르지만 불안정함

포지셔닝: "무지막지한 화력, 극도의 위험"
```

---

### 🐱 Cat Summoner (고양이 소환사) - 귀여움 + 강함

**역할**: 소환수 기반 탱크 | **난이도**: ⭐⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 4  (낮은 직접 공격)
├─ DEF: 6  (중간 방어)
├─ SPD: 5  (느림)
└─ SPT: 13 (매우 높은 지원)

합계: 28점 ✅

핵심 스킬:
├─ "냥이 소환": 고양이 소환 (추가 피해 & 방어)
├─ "냥이 집중 공격": 소환수들의 협력
├─ "9생의 가호": 매 9턴마다 전체 부활
└─ 특징: 개별 스탯은 낮지만 소환수로 강함

포지셔닝: "귀엽지만 치명적"
```

---

### 🥷 Code Ninja (코드 닌자) - 재빠른 해커

**역할**: 속도 + 크리티컬 | **난이도**: ⭐⭐⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 7  (높은 공격)
├─ DEF: 2  (매우 약함)
├─ SPD: 12 (극도로 빠름)
└─ SPT: 7  (좋은 지원)

합계: 28점 ✅

핵심 스킬:
├─ "닌자 습격": 선제 공격 (적을 놀림)
├─ "섀도우 클론": 환상으로 피하기
├─ "쿠나이 투척": 높은 크리티컬 (멀리서)
└─ 특징: 가장 빠르지만 가장 약함 (올인/올아웃)

포지셔닝: "스피드에 모든 것을 건 전사"
```

---

### ✨ Light Mage (빛 마법사) - 빛의 예술

**역할**: 균형 + 회복 | **난이도**: ⭐⭐⭐⭐

```
스탯 배분 (총 28):
├─ ATK: 6  (중간 공격)
├─ DEF: 6  (중간 방어)
├─ SPD: 8  (좋은 속도)
└─ SPT: 8  (좋은 지원)

합계: 28점 ✅

핵심 스킬:
├─ "빛의 검": 공격 + 회복 (동시)
├─ "성역": 모든 피해 75% 감소 (지속)
├─ "부활의 빛": 죽은 파티원 부활 (무제한)
└─ 특징: 가장 균형잡혔고 강력함

포지셔닝: "완벽함을 추구하는 자의 선택"
```

## 4. 조합 시너지

### 🤝 기본 클래스 조합 (2명 멀티플레이)

| 조합 | 시너지 이름 | 보너스 | 설명 |
|------|-----------|--------|------|
| Mage + Knight | "공방일체" | ATK +10%, DEF +10% | 공격과 방어의 완벽한 밸런스 |
| Ranger + Healer | "지원 완벽" | MP -20%, 회복 +20% | 지원과 치유의 조화 |
| Mage + Healer | "마법 시너지" | 마법계 데미지 +15% | 마법사들의 협력 |
| Knight + Ranger | "속도 작전" | SPD +20% | 강함과 빠름의 조화 |
| 같은 클래스 x2 | "완벽 복제" | 스탯 +5% | 같은 클래스 2명 (드물지만 가능) |

## 5. 플레이 스타일별 권장 클래스

### 💪 공격형 플레이어
```
1순위: Frontend Mage (10 ATK, 10 SPD)
     → "먼저 때리고 강하게"

2순위: Code Ninja (7 ATK, 12 SPD)
     → "극도의 속도 + 크리티컬"

3순위: GPU Warlock (8 ATK, 11 SPD)
     → "멀티 타겟 폭탄"
```

### 🛡️ 방어형 플레이어
```
1순위: Backend Knight (10 DEF, 10 SPT)

     → "균형잡힌 생존"
```

### ⚡ 속도형 플레이어

1순위: DevOps Ranger (10 SPD, 7 SPT)

2순위: Code Ninja (12 SPD, 7 SPT)

3순위: GPU Warlock (11 SPD, 7 SPT)

### 💚 지원형 플레이어
